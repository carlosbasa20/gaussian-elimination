# Gaussian Elimination

Python command-line program for applying Gaussian elimination to a matrix.

## What it does

- Reads a matrix from the terminal
- Applies row operations using Gaussian elimination
- Prints the transformed matrix
- Optionally saves the result as a CSV file

## How to run

Run this command in the terminal:

```bash
python main.py
```

## Example input

When the program asks for a matrix, write it like this:

```python
[[2, 1, -1, 8], [-3, -1, 2, -11], [-2, 1, 2, -3]]
```
